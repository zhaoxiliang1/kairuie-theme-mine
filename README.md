# mine — 凯瑞尔电子材料官网 WordPress 主题

**kairuie.com.cn** 自定义 WordPress 主题。

## 说明

本主题为凯瑞尔电子材料（湖北省）有限公司官网定制开发，包含企业产品展示、新闻发布、解决方案等功能模块。

## 文件结构

```
theme-mine/
├── style.css                  # 主题信息
├── index.php                  # 首页
├── functions.php              # 主题功能
├── header.php                 # 页头
├── footer.php                 # 页脚
├── single.php                 # 文章详情
├── about.php                  # 关于我们
├── contact.php                # 联系我们
├── category-news.php          # 新闻列表
├── category-products.php      # 产品列表
├── category-solutions.php     # 解决方案
├── culture.php                # 企业文化
├── honor.php                  # 荣誉资质
├── organize.php               # 组织架构
├── spot-specifications.php    # 规格参数
├── assets/
│   ├── css/                   # 样式文件
│   ├── js/                    # JavaScript
│   ├── images/                # 图片资源
│   └── uploads/               # 上传文件
```
